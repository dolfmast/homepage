# KGA 홈페이지 시안

## 구성
- index.html: 메인 페이지
- styles.css: 디자인 스타일
- script.js: 모바일 메뉴 및 문의 폼 데모

## 실행 방법
1. 폴더를 압축 해제합니다.
2. index.html 파일을 더블클릭하면 브라우저에서 확인할 수 있습니다.
3. 실제 배포 시 카페24, 가비아, AWS S3, Vercel, Netlify 등에 업로드할 수 있습니다.

## 실운영 전 교체 필요
- 대표자명, 사업자등록번호, 주소, 대표번호, 이메일
- 개인정보처리방침, 보험대리점 등록번호, 모집종사자 고지 문구
- 문의폼 서버 연동
- 실제 KGA 로고 및 이미지
